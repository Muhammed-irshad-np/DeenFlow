## 1. Core Design Philosophy

This app is NOT:
- A game
- A dopamine trap
- A guilt machine
- A cluttered marketplace

This app IS:
- A calm spiritual companion
- A daily-use utility
- Minimal, respectful, and trustworthy

No flashy animations.
No loud gradients.

## 2. Emotional Tone

Every screen must feel:

- Calm
- Clean
- Focused
- Non-distracting
- Spiritually grounding

If a screen feels "busy" â€” remove elements.

---

## 3. Color System

### Primary Colors
- Deep Emerald (#0F3D2E)
- Dark Teal (#123C3A)
- Warm Off-White (#F6F4EF)

### Accent
- Gold (used very sparingly for highlights)
- Soft Green for success states

### Forbidden
- Neon colors
- Red warning tones for missed prayers
- Overuse of gold
- Over-saturated gradients

Dark mode must be default.

---

## 4. Typography Rules

### Arabic Text
- Must use proper Uthmani script
- No distortion
- No decorative fonts
- Large readable size
- Generous line spacing

### English / Other Languages
- Clean sans-serif
- High readability
- No condensed fonts
- Minimum 16px for body text

Never shrink Quran text to fit UI.
UI adapts to Quran â€” not the opposite.

---

## 5. Layout Principles

Every screen follows:

- Clear top hierarchy
- Single primary action
- Maximum 2 secondary actions
- No more than 5 core elements per view

White space is mandatory.

---

## 6. Navigation Structure

Bottom Navigation (Max 5 tabs):

1. Home
2. Quran
3. Prayer
4. Tracker
5. Profile

No hidden deep nesting.
Max 3 levels deep.

User should never feel lost.

---

## 7. Quran Screen Rules

Absolutely:
- No ads
- No popups
- No overlay banners
- No subscription interruption

Quran reading must be sacred space.

Audio controls minimal.
Translation toggle simple.
Tafsir in expandable section.

Scrolling must feel smooth and frictionless.

---

## 11. Ramadan Mode UX

When Ramadan is active:

## 12. Performance Rules

App must:
- Launch under 2 seconds
- Work offline for Quran
- Cache prayer times
- Smooth scroll (60fps)

Spiritual apps cannot feel laggy.

## 13. Privacy UX

- No unnecessary permissions
- Location explained clearly
- No background tracking
- Clear privacy statement
